# Glog| Glassic Login Form
*Glassmorphism Login Form*
>A modern and responsive login form built with HTML and CSS, using the glassmorphism design trend. This project focuses on clean UI, soft transparency, blurred backgrounds, and a minimal user experience.

Features Responsive login form `layout`
Glassmorphism card effect
CSS blur and transparency styling
Font Awesome icon support
Clean and beginner-friendly HTML structure
Modern UI suitable for portfolio projects
***Technologies Used***
- HTML5
- CSS3
- Font Awesome
***
Project Structure:

glog/
├── index.html
├── style.css

How to Use
Clone or download the project, then open index.html in your browser.

```html
git clone https://github.com/Naghmehnik87/glog.git
cd glog
open index.html
```
Purpose
This project was created to practice modern frontend design techniques, especially glassmorphism effects using CSS properties such as backdrop-filter, transparency, gradients, shadows, and border styling.
Preview
The form includes a centered login card with input fields, icons, a login button, and supporting links such as “Forgot password?” and “Register”.
